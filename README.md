# Anti-sedentary Reminder  
A product designed to prevent prolonged sitting, using a wearable waist sensor that interacts with a desktop object to visualize the health risks of sitting too long. Through lighting effects and the posture of a small figurine on the desktop piece, the product reminds users to stand up and move.
